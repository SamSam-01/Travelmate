# front

A new Flutter project.

## Environment

Copy `.env.template` to `.env`, then fill:

```env
SUPABASE_URL=
SUPABASE_ANON_KEY=
MAPBOX_ACCESS_TOKEN=
```

Run the app with:

```sh
flutter run --dart-define-from-file=.env
```
